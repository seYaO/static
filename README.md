# staic

## 插件安装
```
yarn install
```

### 打开本地服务
```
yarn serve
```

### 打包touch文件
```
yarn build
```

### 打包上传touch文件
```
yarn deploy
```

### 上传pc文件[配置的最新一个专题]
```
yarn uploadpc
```

### 上传图片
```bash
# 上传pc端图片
# 图片地址 https://img1.40017.cn/cn/s/2020/zt/pc/xxxxx/red.png
yarn uploadimg xxxxx pc

# 上传touch端图片
# 图片地址 https://img1.40017.cn/cn/s/2020/zt/touch/xxxxx/red.png
yarn uploadimg xxxxx touch
```
